Parameters

JSWASM unsigned char* Superpowered::Waveform::getPeakWaveform ( bool  takeOwnership = false)

Returns with 150 points/sec waveform data displaying the peak volume. Each number is an unsigned byte (8-bits), representing one "pixel". Available after calling makeResults().

Parameters
takeOwnershipIf true, you take ownership on the data, so don't forget to free() the memory to prevent memory leaks. Use _aligned_free() on Windows.
